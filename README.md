# study-labs
개발 기초 역량 강화를 위한 데일리 실습 모음 (SQL / Linux / C / Go).
- sql-practice/: 문제-정답(SQL), 스키마/샘플데이터
- linux-lab/: 명령어 실습과 로그
- c-lab/: C 언어 실습
- go-lab/: Go 언어 실습
## Day 1
- SQL: GROUP BY/HAVING/MAX/ORDER BY
- Linux: 시스템 탐색/로그 검색
- C: 구구단 프로그램
- Go: 구구단 프로그램

- SQL: 조인 + 집계 쿼리 작성 가능

- Go: 문자열 파싱 후 단어 빈도수 계산 가능  
- NCS: 기본 유형 문제 풀이 시작

- SQL: 서브쿼리 활용(MAX/AVG), 부서별 최고 급여자/평균 이상 사원, 미주문 고객 조회
